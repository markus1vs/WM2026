# WM 2026

GitHub-Pages-Paket für den WM 2026 Simulator.

## Dateien hochladen

Lade alle Dateien direkt in den Hauptordner deines GitHub-Repositories hoch:

- index.html
- manifest.json
- apple-touch-icon.png
- icon-192.png
- icon-512.png
- README.md
- .nojekyll

## GitHub Pages

Settings → Pages → Deploy from a branch → main → /root → Save

Danach ist die App erreichbar unter:

https://DEINNAME.github.io/REPO-NAME/

Bei Markus aktuell vermutlich:

https://markus1vs.github.io/WM2026/

## Neue JSON laden

Wenn du von ChatGPT eine neue JSON-Datei bekommst:

1. App öffnen
2. „Update-Datei laden“ drücken
3. JSON auswählen

Das Programm muss dafür nicht jedes Mal ersetzt werden.
